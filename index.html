Pada pembelajaran sebelumnya kita mempelajari tentang Dasar Dasar HTML Adapun Ringkasannya yaitu;

HTML adalah bahasa komputer yang digunakan untuk membuat kerangka atau struktur untuk halaman website diinternet.Adapun selain HTML juga ada CSS dan JavaScript,fungsi HTML adalah sebagai kerangka yang memberi struktur pada website. fungsi CSS yakni sebagai tampilan yang memberi warna dan layout pada website,Sedangkan fungsi JavaScript adalah sebagai element yang membuat website menjadi interaktif.

HTML terdiri dari beberapa komponen salah satunya HTML Tag,pada umumnya ada 2 tipe html tag yaitu opening tag(tag pembuka) dan closing tag(tag penutup), html opening tag ini digunakan untuk menampilkan paragraf.Selain itu juga ada HTML Element,HTML Element merupakan sebuah komponen dalam halaman web bisa berupa paragraf,judul atau gambar.Ada 2 jenis HTML Element yaitu HTML Element yang memiliki Opening tag dan closing tag,ada juga Empty HTML Element yang memiliki self closing tag yang hanya memiliki opening tag dengan garis miring sebelum kurung tutup,contohnya br atau img.Tag br digunakan untuk berganti baris atau spasi,sedangkan tag img digunakan untuk menampilkan element gambar.
